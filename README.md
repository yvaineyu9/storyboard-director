# 分镜导演 Agent

面向带屏 AI 眼镜（AIUI Runtime）的语音分镜助手。

> 用户用一句话语音描述「想拍什么画面 + 想表达什么情绪」，系统解析意图、并行分析、组合输出一套
> 可执行的分镜方案，并把每个分镜的构图与剪辑节奏综合成一张**两层色剪辑分镜示意图**返回。

## 一句话流程

```
语音描述 → 分析意图(可追问) → 并行[画面→构图 | 情绪→剪辑节奏] → 组合 → 两层色剪辑分镜示意图 → 分镜指引
```
