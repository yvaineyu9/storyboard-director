# 模块B · 剪辑节奏

把三份博主分镜分析素材里的剪辑节奏规律整理成 **「情绪 → 剪辑节奏」映射**。管线(b)「情绪 → 剪辑节奏」调用本模块，为每镜给出时长、切换速度与转场偏好。

---

## 从素材提炼的节奏规律

- **全局速度**：A《I Just Turned 20》约 87 BPM，B《做自己人生主角》约 79 BPM —— 中速偏抒情。
- **快切 montage**：0.5–0.6s / 镜，切点精确卡在音乐 build-up / drop（A 两个能量峰值）。
- **口播长镜**：40–137s（A）、58s / 27s（B）—— 慢下来让观点沉淀。
- **留白 → 爆发**：每个高潮前先插一段静音 / 降噪（1:10、8:10、14:40）再砸节拍。
- **生活流 b-roll**：0.5–1s / 镜，卡歌词节拍（B 夜聚）。
- **音乐与情绪错位**：动靠音乐、静靠留白（B：动势段配乐推满，情绪核心反而压低音乐留人声）。

## 字段约定（供组合层使用）

每镜由本模块给出：

- `duration`：该镜时长（秒）—— 决定示意图「节奏条」长度
- `cutSpeed`：切换速度档位（慢 / 中 / 快 / 极快）—— 决定节奏条密度
- `transition`：到下一镜的转场（硬切 / 叠化 / 淡入淡出 / 匹配剪辑 / 甩切 / J切 / L切）
- `pause`：是否在本镜前安排留白蓄势（用于高潮前）

> 受短样片 3~6 镜约束：节奏映射输出的镜数需收敛到 3~6，长镜场景取代表镜、快切场景做代表性抽样。
