## 四、遗留缺陷（单列）

1. **【真缺陷 · 已知】bisect 二分构图真库 0 覆盖**
   - `lib/figure.js:53` `SUPPORTED_COMPOSITIONS = [...MODULE_A_COMPOSITIONS, 'bisect']`，figure 能画 bisect、vocab 收录，但 `MODULE_A_COMPOSITIONS`（10 种）**不含 bisect**，a0-validate 也不要求其覆盖（`a0-validate.js:150-152` 只断言 MODULE_A 的 10 种 ≥1）。
   - 实测 `a0-validate` 输出 `composition coverage ... "bisect":0`：24 条真库**无任何模板用 bisect**，A2 生产中永远选不到二分构图。
   - 影响：非阻断（仍有 10 种构图可用），但「二分法」构图能力在生产链路缺失。若需补，重跑 A0 引导环节增补 bisect 模板即可（离线维护，不在生产流程）。

2. **【真机待验】以下只能在 AIUI 真机/眼镜上验证，离线脚本无法覆盖：**
   - UI 实际渲染（streamdown 卡片 / canvas 胶片条 / scroll-view / 480px 布局 / error-state）。
   - canvas N=3 与 N=6 的 backing 尺寸表现（当前固定 894×190 最大 backing，N<6 左对齐留黑底，真机需目检无截断/拉伸）。
   - 端侧 LanguageModel 真实推理质量（A1 抽取准确度、追问话术、A2 选模板合理性、A3 节奏合理性）—— 离线只验证了降级与契约，未验证真实 LLM 输出质量。
   - SpeechRecognition 真实收音 / interim 转写 / zh-CN / 硬件键切麦，只能 AIUI 真机验。
